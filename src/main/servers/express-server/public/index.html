Express Server Example

Welcome to the Express Server example page. This simple server demonstrates how to use Express.js to create a web server with REST API capabilities.

Available Endpoints:

GET /

Returns this welcome page.

GET /api/hello

Returns a simple JSON response with a greeting message.

Example response: { "message": "Hello from Express Server API!" }

GET /api/users/:id

Returns details for a specific user based on the provided ID parameter.

Example response for /api/users/123: { "userId": "123", "message": "Fetching details for user 123" }

POST /api/data

Accepts JSON data in the request body and returns a confirmation with the received data.

Example request body: { "name": "John", "age": 30 }

Example response: { "message": "Data received successfully", "receivedData": { "name": "John", "age": 30 } }
